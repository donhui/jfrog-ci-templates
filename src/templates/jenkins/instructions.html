Next steps
1. Log into your Jenkins instance.
2. Make sure the Jenkins Artifactory Plugin is installed.
3. Make sure the go client is included in the Jenkins agent's PATH.
4. Configure your JFrog Platform details in Manage Jenkins | Configure System.
5. Replace the in the Jenkinsfile template JFROG_INSTANCE with the ID of the JFrog Platform instance you configured.
6. Push the Jenkinsfile to the root of your git project.
7. Create a Pipelines Jenkins job which uses the Jenkinsfile.
