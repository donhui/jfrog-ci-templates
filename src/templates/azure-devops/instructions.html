Next steps
1. Log into Azure DevOps.
2. Make sure the Artifactory Azure DevOps extension is installed.
3. Configure your JFrog Artifactory instance as a Service Connection in Azure DevOps.
4. In the azure-pipelines.yml file, replace JFROG_INSTANCE with the JFrog Artifactory Service Connection ID you configured in Azure DevOps.
5. Create a new pipeline in Azure DevOps which uses the azure-pipelines.yml file.

Additional information
The CI templates are hosted in the github.com/jfrog/jfrog-ci-templates repository.
Read more about the Artifactory Azure DevOps extension used in the template.
