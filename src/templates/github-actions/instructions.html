Next steps
1. Make sure JFrog CLI is installed on your local machine by running 'jfrog -v'.
2. Configure the details of your JFrog Platform by running 'jfrog c add'.
3. Export the details of the JFrog Platform you configured, using the server ID you chose. Do this by running 'jfrog c export '.
4. Copy the generated token to the clipboard and save it as a secret on GitHub.
5. In the workflow.yml file replace JFROG_INSTANCE with the server ID you configured.
6. Push the workflow.yml to your git project under .github/workflows/

Additional information
The CI templates are hosted in the github.com/jfrog/jfrog-ci-templates repository.
Read more about the Setup JFrog CLI GitHub Action used in the template.
